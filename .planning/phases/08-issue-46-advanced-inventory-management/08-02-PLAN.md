## Implementation Notes (2026-04-20)
- Added authenticated, tenant-scoped inventory APIs in `InventoryController`:
  - `POST /inventory/items/:itemId/transactions`
  - `GET /inventory/items/:itemId/transactions`
  - `GET /inventory/items/:itemId/transactions/reconciliation`
  - `POST /inventory/items/:itemId/transactions/:transactionId/reverse`
- Enforced authorization through `JwtAuthGuard + TenantGuard + RolesGuard` with `@Roles(shop_admin, super_admin)`.
- Added DTO validation rules for create/list/reverse transaction payloads, including deterministic adjustment validation.
- Added/updated service tests for invalid transition scenarios and adjustment contract enforcement.
