# Phase 1 Plan 01: Inventory Lifecycle Hardening

## Objective
- **What:** Lock down item lifecycle flows (create, update, soft delete, filter) and category metadata integrity.
- **Why:** Inventory is the foundation for every downstream phase (media, public catalog, AI/social).
- **Output:** Reliable admin CRUD flow with deterministic server-side validation and consistent list/detail behavior.
