# Issue #13 - Pre-Order Management

## Goal
Them chuc nang quan ly danh sach mo hinh pre-order.

## Scope
- DB pre-order, API CRUD, admin pages va public mobile pre-order pages cho pre-order.

## Tasks
1. Tao model `pre_orders` va cac trang thai.
2. Them API tao/sua/huy/chot pre-order.
3. Theo doi coc, so luong, ngay du kien giao.
4. Tao UI admin danh sach + bo loc trang thai.
5. Tao public mobile UI "Mo hinh Dat truoc" theo baseline card layout (badge, countdown, gia, CTA, bottom navigation).
6. Tao public mobile UI "Don hang cua toi" de theo doi trang thai don (dang cho/da ve/da thanh toan) va CTA hanh dong.
7. Tao admin mobile UI "Tao Pre-Order Moi" voi form day du truong du lieu + upload anh.
8. Tao admin mobile UI "Quan ly Pre-order" voi tong quan chien dich, danh sach nguoi dung, va thao tac chi tiet.
9. Lien ket item voi pre-order de dong bo ton kho.
10. Them test cho state transition va smoke test cac man hinh mobile MVP.

## Deliverables
- Full flow pre-order tu tao den hoan tat.

## Done Criteria
- Theo doi duoc so luong dat truoc va tien do xu ly.
- Khong xung dot voi luong item thong thuong.
- MVP co giao dien public mobile pre-order toi thieu theo baseline duoc cung cap.
- MVP co bo man hinh mobile day du: "Mo hinh Dat truoc", "Don hang cua toi", "Tao Pre-Order Moi", "Quan ly Pre-order".
